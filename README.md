# fourohfour.vim

> lookup HTTP status codes without leaving Vim

Can't seem to remember what that `408` is, try `:FOF 408` to be reminded
that it's `Request Timeout`.

Want to see a list of all 5xx status codes, give `:FOF 5xx` a whirl.
